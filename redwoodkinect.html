Redwood City Kinect Storefront Interactive

Redwood City Improvement Association wanted to start drawing more attention to certain neglected / empty store fronts and had asked the Fresh AV team to come up with a creative solution. Working together we created a projection setup that utilizes a kinect to do various motion tracking spectacles. There are several modes that can be selected, for some modes it uses the skeletal tracking features of the kinect and for others it merely uses fun camera effects. The Fresh AV team selects the mode for a certain store front and then that mode runs for about a month or so until it is moved to a new area. I designed and built all the computer servers that run the spectacles (four machines in total) and also did all the custom Touchdesigner software programming. The computers launch and close the program using Windows Task Scheduler and Batch files, so the entire system is automated.
